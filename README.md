## 문제풀이 원칙
1) 최소 30분 최대 50분은 알고리즘 설계를 한다.<br>
2) 코드는 최대 1시간 안에 작성한다.<br>
3) 반례 케이스는 답부터 보지 말고 스스로 고민해본다.<br>
4) 2시간 안에 답을 못 맞출 경우 해답을 보고 공부한다.<br>
5) 틀렸던 문제는 시간 날 때마다 공부해본다.<br>
